# Section4-1

## 家計簿アプリで最低限やりたいこと

- 取引（入金/出金）を登録・一覧表示・詳細表示・編集・削除したい
- 月次集計（収入合計/支出合計/差額）を見たい
- カテゴリ（食費、家賃…）を選びたい

但し、RESTの「リソース」の観点を考慮しDBに保存される“もの”を中心に設計する


## リソース候補

- transactions(取引)
- categories(カテゴリ)

## 項目

### 何を管理するか

- transactions　いつ、何に、いくら
- categories　給与、食費、交通費、副収入、娯楽、その他（その他はメモ記入したい）
  
### 関係性

transactions：categories　N:1
